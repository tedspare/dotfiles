# Setup

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install gh
gh auth login
git clone https://github.com/tedspare/dotfiles.git ./.dotfiles
cd ./.dotfiles
chmod +x ./setup.sh
./setup.sh
```
